-----------------------------------------------------------------
Angular Core Concepts
-----------------------------------------------------------------

Directives

    3 types

    1. component
    2. structural
        => to change component's template structure
    3. attribute
        => to change element's properties


Data Binding

    3 types

    1. interpolation
        syntax: <element>{{expression}}</element>
    2. property
        syntax: <element [prop1]="expr" [prop2]="expr"/>
    3. event
        syntax: <element (event-name)="handler($event)"/>
